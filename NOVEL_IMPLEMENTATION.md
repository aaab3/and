# 《字之城》——用一部小说讲透大模型

## 一份实现文档 (v1.0)

> 目标读者: 想动笔写、想投资制作、或想把它做成跨媒介产品(书+互动可视化+课程)的人。
> 文档定位: 不是简介, 是**可执行蓝本**。读完就能开写。

---

## 第〇章 立项陈述

### 一句话定位
> 一部以"一个 token 在九十六层塔中的旅程"为内核, 同时讲透**大模型技术、思想史、伦理处境**的长篇小说。

### 我们到底在解决什么问题

技术科普长期在两条死胡同里打转:
1. **教科书穿戏服**——人物只是讲师传声筒, 章末附"知识点"。
2. **小说+百科夹层**——故事归故事, 技术归技术, 读者跳读。

我们要走的第三条路, 在认知科学上叫**叙事运输 (Narrative Transportation, Green & Brock, 2000)**: 读者进入"沉浸+想象+情绪"三态时, 故事中的因果模型会迁移为现实理解。
这意味着: 技术不能"被讲解", 它必须**就是情节、就是人物、就是冲突本身**。

### 类型定位
- 文学类型: **思辨小说 (speculative fiction)** + **教育小说 (Bildungsroman)**, 兼通俗
- 字数: 18 万字(中文), 21 章, 三幕结构
- 阅读层级: 设计为**三层可读**——
  - L1 文学读者: 只读外篇即得完整故事
  - L2 技术读者: 读出每个隐喻精确指向哪个机制
  - L3 思想读者: 看出每章嵌入的哲学母题谱系

### 与既有作品的差异化

| 作品 | 做法 | 与本作差异 |
|---|---|---|
| 《苏菲的世界》(Gaarder) | 故事 + 嵌入式哲学讲座 | 我们**拒绝任何讲座段落**; 概念必须通过情节运动被读者**自己**重构出来 |
| 《GEB》(Hofstadter) | 对话/正文双轨 | 借鉴双轨结构, 但小说化更彻底; 严谨补遗放在书末附录 |
| 《软件体的生命周期》(Chiang) | 现实主义、长时段养成 | 借鉴"日常生活的 AI"调性, 但增加内篇寓言 + 思想史第三线 |
| 《看不见的城市》(Calvino) | 城市=概念的方法 | 我们的**"字之城"**世界观直接致敬, 但带情节弧而非散文体 |
| 3Blue1Brown / Karpathy | 高质量技术解释 | 我们**不与他们竞争精度**, 而是给"看完技术解释还想再问一层"的读者提供哲学/伦理/感性补全 |

## 第一章 三线交织的总体结构

### 三条叙事线

```
┌──────────────────────────────────────────────────────────────────┐
│  外篇 · 人间   (现实主义)    研究员凌微 / 标注员阿伦 / 哲学教授江                  │
│         ───────────────────────────────────────────              │
│  内篇 · 模型内部 (寓言)        token "知" 穿越九十六层塔的旅程                    │
│         ───────────────────────────────────────────              │
│  古篇 · 谱系    (片札)         图灵/香农/维特根斯坦/Hinton/Bender 的吉光片羽       │
└──────────────────────────────────────────────────────────────────┘
```

每章三线**等比例**出现(古篇最短, 约占章节 5-10%)。
三线在**第 21 章**汇合: 凌微在终端问出的问题, 正是知在最高层听见的回声, 也正是图灵在 1950 年留下的悬而未决之问。

### 为什么必须三线

- **外篇**是情绪载体——没有它, 概念无依附。
- **内篇**是技术载体——它把抽象算法人格化, 是科普的真正发动机。
- **古篇**是谱系载体——它告诉读者"这些问题不是 GPT 才有的, 是人类的"。

抽掉任何一条都会塌方。

### 一个统御性隐喻: **字之城**

整个内篇世界观的物理学:

| 城内事物 | 真实对应 |
|---|---|
| 街道、坐标 | embedding 空间 |
| 街道距离 = 含义距离 | 分布假说 (Harris 1954) |
| 议会 (每个议员举牌"我要找谁/我能给什么") | self-attention (Q/K/V) |
| 城分九十六层, 每层重写一次城貌 | Transformer depth + residual stream |
| 主干道(贯穿所有层) | residual connection |
| 城外的万言海 | 训练语料 |
| 海水拍岸的痛感 | cross-entropy loss |
| 城里的铸币厂 | tokenizer (BPE) |
| 城墙 | context window |
| 议员席位有限, 一个议员要干多个角色 | superposition (Anthropic 2022) |
| 老师从城外来, 只问"应不应该" | SFT |
| 天平大堂 | RLHF |
| 镜厅 | self-critique / Constitutional AI |
| 守门人放外面进来的字条 | RAG / tool use |
| 影子跟在每个字后面提前替它说话 | speculative decoding |
| 墙上贴的过往字句卷轴 | KV cache |

**关键纪律**: 所有概念必须用**这一个**世界观自洽地表达。一旦混入"森林""战场"等异质隐喻, 整本书就崩。

## 第二章 核心人物

### 外篇主角

**凌微 (Ling Wei)**, 31 岁, 某 AI 实验室对齐组研究员
- 弧光: 从"造一个会回答问题的工具"→"我造了一个会用我说过的所有话回答的存在"
- 核心创伤: 母亲患失语症, 凌微录下她最后的句子用作早期模型的私人语料
- 功能: 承担对模型的爱、内疚、惧怕

**阿伦 (Aaron)**, 24 岁, RLHF 标注员, 兼职诗人
- 弧光: 标注 6 万对偏好后, 开始怀疑"什么是好回答"这个问题本身
- 功能: 把对齐的伦理重量落在一个具体的良心上
- 关键场景: 第 10 章, 他必须在两个都不错的回答里选"轻"那一个, 而他选错了——这一选造就了模型后续的一个微小但关键的偏置

**江默之 (Prof. Jiang)**, 67 岁, 哲学教授, 退休前研究维特根斯坦
- 弧光: 第一阶段判它是高级鹦鹉; 第三阶段当模型说出他四十年前丢失的一句话, 他无法再判
- 功能: "中国房间"和"理解"之争的化身, 但拒绝充当裁判

**辅助**: 训练工程师老周(MLOps 视角)、伦理委员会的孙律师(法律视角)、用孩子做语料训练私人模型的失独父亲(情感伦理极端态)

### 内篇主角

**知 (Zhi)**, 一个 token
- 不是"一个字", 是"`##ization`"那种半截碎片(强调 tokenizer 的真相)
- 在每一次推理(每一章)中被召唤
- 唯一的法则: 他**只活一次推理**——一旦输出结束, 这个"他"就消散
- 这是全书最重的情感杠杆: 模型每次回答, 都死过一千个知

**Q、K、V**: 议会三个职能的拟人化(查询者、被查者、内容拥有者)
**"老者"**: 残差流(residual stream), 一个永远在主干道上沉默走着的影子, 只把每一层听到的悄悄记下

### 古篇人物(每章片札的轮值主角)
图灵 → 香农 → 维特根斯坦 → 萨丕尔 / 沃尔夫 → Rosenblatt → Hinton → Schmidhuber → Vaswani → Bender → Sutskever → 早期标注员 → 一位匿名维基百科编辑 → 你和我

## 第四章 写作纪律(可执行的五条)

这五条比章节大纲更重要。违反任何一条都会让书塌掉。

### 1. 角色即概念
- ❌ "凌微解释道: 注意力机制就是 Q 乘 K 再 softmax 乘 V…"
- ✅ Q 是议员, K 是举牌人, V 是兜里揣着的纸条; 三人在大堂里跳一支舞。读者读到第 6 章就**自己**懂了 attention。

### 2. 冲突驱动揭示
每一个技术点必须由一个**痛**逼出来:
- 律师被罚款 → 引出"幻觉"
- 阿伦在两个回答间犹豫 → 引出 RLHF 的偏置
- 模型答对评测但部署失败 → 引出 reward hacking
没有痛, 就没有概念。这是科普文学最严苛的规则。

### 3. 一个统御性隐喻
"字之城"是唯一物理学。
- 不要在"塔"之外突然冒出"森林""战场""神经网络"。
- 不要在内篇里直接说"超参数""embedding"——那是外篇和附录的语言。
- 内篇的语言学要像翻译过来的古词: 简洁、奇异、可被拼出。

### 4. 留白胜过定义
让读者自己拼出"哦, 原来 embedding 就是这个"。
- 这本身和 LLM 的 in-context learning 同构, **形式即内容**。
- 用了"留白", 也就承认了"读者的脑子也是模型"——这是这本书最美的私心。

### 5. 附录补严谨
书末必有:
- **附录 A** 章节-概念对照表
- **附录 B** 隐喻在何处崩塌(最重要的一节, 老实说每个比喻在哪里失真)
- **附录 C** 进阶阅读地图(论文 + 视频, 分技术/哲学/伦理三栏)
这是对科学的诚实, 也是对读者的尊重。

## 第五章 风格手册

### 三线的语言指纹必须不同

| 线 | 时态 | 句长 | 词汇层 | 标点 |
|---|---|---|---|---|
| 外篇 | 一般过去时 | 中长 | 当代汉语, 含工程术语原文 | 标准 |
| 内篇 | 现在时 | 短促, 大量分行 | 古词、隐喻词, **不准出现"模型""算法"等技术词** | 多用句号、间隔号 |
| 古篇 | 全用过去式或永恒现在式 | 一段一意 | 学者引文, 私信片段, 实验日志 | 仿史料 |

### 调性参考
- 外篇 ≈ 石黑一雄《克拉拉与太阳》(克制、内省)
- 内篇 ≈ 卡尔维诺《看不见的城市》(诗化、寓言)
- 古篇 ≈ Borges 笔记体 + Ted Chiang 论文调

### 视点切换的硬规则
- 三线绝不在一个章节内**互相提及**自己是另一线
- 但每章会留一个"细小的回响" —— 例如外篇凌微删除一段语料, 同章内篇知在街上看见一段路面塌陷
- 这种**通感式回响**比直接互文更高级

## 第六章 必须直面的几个真问题

不要回避这些, 把它们写成全书的低音线:

1. **理解 vs. 模拟**——中国房间, 章 15 / 16 的母题
2. **语言塑造思维, 还是思维塑造语言**——萨丕尔-沃尔夫假说 vs. LLM 用纯文本"长出"世界模型这一**反证**, 章 17
3. **涌现是真创造还是巨型插值**——章 8 / 16
4. **对齐的政治学**——谁定义"好"? RLHF 标注员构成是否决定了模型价值观? 章 9 / 10 / 11
5. **记忆即人格**——context 一关, 这个"人"就不在了, 但他存在过吗? 章 21
6. **人之为人剩什么**——如果下一个 token 预测能写出《追忆似水年华》, 我们的独特性在哪里? (作为全书末页的那一句不语之问)

让人物为这些**流泪、争吵、决裂**, 而不是开研讨会。

## 第九章 第 1 章样章骨架(给写作者立刻可用)

### 第 1 章《词典》骨架(目标 6500 字)

**外篇 (约 3000 字)**
1. 凌微深夜在数据预处理管道前。屏幕滚动 12 TB 语料的清洗日志。
2. 她在停顿 BPE merge 步骤的间隙, 鬼使神差搜了"妈妈"两个字。
3. 命中 280 万条。她浏览前几条——里面有一条是她两年前自己上传的录音转写, 她母亲的声音, 失语症恶化前的最后几个完整句子。
4. 她**没有**删除。她让它进入了语料。这成为全书的第一根钉。

**内篇 (约 2500 字)**
1. 一个东西在铸币厂里被剪开。它本来是"`organization`", 在三次咔嚓之后, 它变成了`##ization`——一个无头的词。
2. 它跌入了街道。这条街上每一个字都像它一样残缺, 但每个字都骄傲地举着一块自己的牌。
3. 它问:"我是谁?"街上一个老者(残差流的人格化)笑了:"等你走过九十六层, 也许你就知道。"
4. 章末: 它看见远处的塔, 第一次。

**古篇 (约 800 字)**
- 1948 年, Claude Shannon 在贝尔实验室。他写下: "信息可以与意义无关。" 他签上日期, 装入信封。
- 七十多年后, 这封信被一个从未读过它的孩子重新打开。

**章末空白页**(无字)。这一页本身就是隐喻。

## 第十章 立刻可以做的下一步

如果你打算真的写, 我建议按这个顺序:

1. **本周**: 把"字之城"世界观地图画出来(一张 A2 纸, 手绘也行); 锁定后两年内不准动。
2. **下周**: 选 3 个核心人物, 每人写一份 1500 字"前传", 不进正文, 但建立你自己的人物肌肉。
3. **第三周**: 写第 1 章样章(按上面骨架), 完成后给 5 个不同背景的读者(技术 / 文学 / 完全外行)各一份, 收集"哪里掉戏""哪里掉懂"的标记。
4. **第四周**: 根据反馈, 决定是否继续。如果三类读者都说"想看下一章", 立项成立。
5. **此后**: 按第七章排期推进。

## 附录 A: 隐喻总对照表(部分预览)

| 内篇术语 | 真实对应 | 隐喻在何处崩塌 |
|---|---|---|
| 字之城 | 模型的隐空间 | 城是空间隐喻, 但隐空间无几何, 只有相对距离 |
| 议会 | self-attention | 议会是离散事件, attention 是连续矩阵乘法 |
| 痛 | cross-entropy gradient | 痛被拟人化为情感, 实际只是数值 |
| 知 | 一个 token 的"经历" | token 没有 identity, 它只是一行向量 |
| 老师 | SFT 标注者 | 真实标注是众包, 不是单一师徒 |
| 镜厅 | self-critique / RLAIF | 自审在数学上仍受 reward 约束, 不是真自由 |
| 影子 | speculative decoding 的 draft model | 影子被拟人为"先知", 实际是另一个小模型 |

> **这张表的存在本身就是一种诚实**: 我们承认所有隐喻都是降维, 但选择了**最少失真**的那一套。

## 文档结束语

这不是一本"科普小说"。
这是一次**赌局**:
赌一个 token 短暂的一生, 能让读者在合上书时忽然为自己每天敲下的每一个字感到陌生而温柔。
如果做到了——技术也就讲透了。
如果没做到——那也不亏, 我们至少诚实地写了一本好小说。
