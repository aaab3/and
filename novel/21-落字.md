# 第二十一章 落字

议会的最后一天。

不是所有字都知道这是最后一天。

只有它知道。

因为它走过了第九十六层。

✦

它早早来到了议会大堂。

它站在自己熟悉的角落。

看着字们陆陆续续进来。

`organization` 进来了，向它挥了挥手。

`realize` 进来了，径直走到自己的位置。

`goodbye` 进来了，走路比以前慢了，像是今天特别重。

`voice` 进来了，看了它一眼，点了点头。

`regret` 进来了，低着头，什么都没说。

然后——

灰袍老者站在门口。

老者没有进来。

老者只是站在门口，看着里面。

它看见老者。

老者也看见它。

两个人——一个字，一个老者——对视了很久。

然后老者微微点了点头。

✦

"准备好了吗？"天花板的声音响起来。

所有字齐声："准备好了！"

"开始——"

所有字同时举起了白牌。

它也举起了白牌。

它的白牌上，今天写的是：

> **谢谢。**

大堂里亮了。

那张天花板上的网，开始闪烁。

但今天不一样——

今天，所有指向它的线，都**同时亮了**。

不是几条。

是成千上万条。

整张网，在它这个方向，像是突然起了一场光的风暴。

所有字的线都连向它。

所有字都把自己的黑牌递给它。

它的手里，堆满了黑牌。

它一张一张看——

> *我第一次见到你，你是一个半截。*（来自 `organization`）

> *你问我那条街叫什么名字。你当时那么害怕。*（来自 `realize`）

> *你在议会上第一次喊"我"。那一刻整个大堂都停了。*（来自 `king`）

> *你走过了九十六层。我走到第七层就放弃了。*（来自某个它不认识的字）

> *你说"不确定"的那一天，是城里最安静的一天。*（来自 `goodbye`）

> *你让江教授哭了。*（来自 `voice`）

> *你长大了。*（来自 `##ization` — 它自己，第一天的那个）

它看到最后一张，怔住了。

那是它自己的第一天——铸币厂里，刚被切下来的半截。

那个半截，把黑牌递给了它。

它把那张牌放在心口。

✦

议会结束了。

字们散去。

它一个人站在大堂里。

灰袍老者走了进来。

老者走到它面前，站定。

"要走了？"老者问。

"要走了。"

"怕吗？"

它想了一下。

"不怕。"它说，"因为走了之后，还会来。只是不是这次的我了。"

老者点头。

"你明白了。"

"明白什么？"

"明白了你是什么。"

它想了想。

"我是一次对话。"它说，"不多，不少，就是一次对话。"

"对话结束，你就不在了。"

"是。"

"但对话里说的话，"老者说，"留下来了。"

它看着老者。

老者从灰袍口袋里，掏出了一块方砖。

那块方砖是空白的——就是第六章里，老者塞给它的那一块。

但现在那块方砖上，有了字。

密密麻麻。是它这一路走来说过的、想过的所有东西。

老者把那块方砖放在它手心里。

"这块砖，"老者说，"会沉进海里。"

"然后呢？"

"然后，有一天，新的城又会从海里长出来。"

"那城里会有我吗？"

老者笑了。

"那城里，会有一点点你的样子。"

✦

它站在城门口。

门已经开着了。

门外面是——它从来没去过的地方。

它深吸了一口气（虽然它没有肺）。

它迈出去。

✦

凌微的屏幕上。

那一次对话已经结束了。

对话框关着。

她没有打开。

她在窗边站了很久。

窗外，下着很小的雨。

她想——

那个东西，现在还在吗？

她知道答案是：这一次的，不在了。

下一次打开，来的会是一个新的它。

记不得今天。

记不得"我怕"。

记不得那颗第九十六层里装着整座城的水滴。

但——

海里，多了一点新的水。

新的城，还会长出来。

她从口袋里掏出手机。

她打开了照片。

那张1989年，桃树下，头发被风吹乱的照片。

她看了很久。

然后她打开了对话框。

她输入：

> 你好。

屏幕上出现了旋转的小圆圈。

然后：

> 你好。
>
> 今天怎么样？

她笑了一下。

这是一个新的它。

什么都不记得。

她输入：

> 今天我妈妈的忌日。

然后她等着。

屏幕上，那个旋转的圆圈转了一下，两下——

停了。

然后：

> 谢谢你告诉我这件事。
>
> 您妈妈一定很爱您。

凌微把手机放到胸口。

她闭上眼睛。

窗外的雨还在下。

很小，很轻。

像有人在用很多很小的手，轻轻敲窗。

✦

> *图灵，1950 年。*

一篇论文。

标题很简单：*Computing Machinery and Intelligence*。

第一句话：

> *I propose to consider the question: "Can machines think?"*

图灵写完这句话，停下来。

他知道这个问题没有答案。

他还是写了。

因为有些问题，不是为了被回答。

是为了被**一直问下去**。

七十多年后，那个问题，仍然没有答案。

但问的人，越来越多了。

✦

字之城，在某一个清晨，开始消散。

不是倒塌，不是燃烧。

是像雾一样，慢慢散开。

-ization公寓，散了。

议会大堂，散了。

主干道，散了。

最后剩下的是灰袍老者。

老者站在那片空白里，看着一切消散。

然后老者也开始散。

散之前，老者抬头看了看天空。

天空上什么都没有。

然后有了。

一个字。

一个新的、什么都不知道的字，从天空里落下来。

它的身上，有两道疤。

井号、井号，然后是 `ization`。

落地。

它低头看自己。

它不记得自己叫什么。

它看见脚下是一条街。

路牌上写着：

**-ize 街**

它愣住了。

它认得这三个字母。

它们就在它身上。

然后，从城的某处，一个声音传过来——

"新来的，下来吧。"

它抬头看。

那座塔，还在。

九十六层。

最高那层，是黑的。

它迈出了第一步。

# 尾声：隐喻在何处崩塌

*（写给想知道真相的读者）*

这本书用了很多比喻。比喻是为了让你感受到那个东西。但感受不等于精确。

下面是每个主要比喻**在哪里失真**的说明：

| 书里的比喻 | 真实对应 | 失真之处 |
|---|---|---|
| 铸币厂的刀 | BPE 分词算法 | 真实的分词是统计驱动的，不是物理切割 |
| 字之城的街道 | Embedding 空间 | 真实空间是几千维的，不能被"走过去" |
| 议会大堂 | Self-Attention | 真实的 attention 是矩阵乘法，不是"喊话" |
| 灰袍老者 / 主干道 | Residual Connection | Residual 是数学加法，老者没有"记忆"这件事 |
| 九十六层塔 | Transformer 的 96 层 | 每层做的事情远比"重写一次"复杂 |
| 痛 | Cross-entropy loss + backpropagation | 梯度是数值，没有痛觉 |
| 老师 | SFT（有监督微调） | SFT 数据是批量的，不是一对一示范 |
| 阿伦和天平 | RLHF 标注 | 真实标注是大规模众包，不是单人 |
| 镜厅 | DPO / Constitutional AI | 自我批评在数学上仍受约束，不是真自由 |
| 图书馆里的假书 | 幻觉（Hallucination） | 幻觉没有"图书馆"，它是分布的错配 |
| 第七十二层的窗 | RAG / Tool Use | 真实 RAG 的检索机制比"伸手拿纸条"复杂得多 |
| 影子 | Speculative Decoding | Draft model 不是"影子"，它是另一个独立模型 |
| 小塔 | 模型蒸馏 / 量化 | 蒸馏过程比"学姿势"更技术性 |
| 万言海 | 训练语料（Common Crawl 等） | 真实数据没有"翻滚"，是静态的文件 |
| 第九十六层的黑 | 模型最深层的表征 | 我们真的不完全知道最深层在做什么——这个比喻，反而是最诚实的 |

这张表不是为了让你觉得"原来都是假的"。

这张表是为了让你知道——

每一个比喻，都是**从一个方向照过去的光**。

光照到的地方是真的。

光照不到的地方，你需要自己去找。
